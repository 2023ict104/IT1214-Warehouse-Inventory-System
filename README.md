# IT1214 Warehouse Inventory System

## Project Description
This project is a Java-based Warehouse Inventory System.
It helps manage products, track stock levels, and update inventory efficiently.

## How to Run the Program
1. Install Java (JDK 8 or higher)
2. Open terminal / command prompt
3. Compile the program:
   javac warehouse.java
4. Run the program:
   java warehouse
